# recorder
## 说明
离线录音机，使用浏览器存储，可离线保存数据，保证数据不丢失，免去在线录音机刷新丢失所有数据的问题，实现自动缓存并防止录音失败后丢失数据（out of memory等）
## 功能计划
- [x] 录音
- [x] 播放器
- [x] 播放列表
- [ ] 云存储
- [ ] 实时缓存
- [ ] 自适应布局
